WhatsApp söhbətlərini oxunaqlı HTML formatına çevirən Windows tətbiqi.

## Xüsusiyyətlər

- WhatsApp ZIP ixracını HTML-ə çevirir
- Media fayllarını saxlayır (şəkil, video, audio, sənəd)
- Tarix aralığına görə süzgəc
- Tərəf adlarını və nömrələrini redaktə etmək imkanı
- iOS və Android formatlarını dəstəkləyir (Azərbaycan dili daxil)
- Avtomatik yeniləmə

## Tələblər

- Windows 10 (1903) və ya daha yüksək
- x64 və ya ARM64

## Quraşdırma

1. [Son versiyadan](https://github.com/azadaydinli/wa-export/releases/latest) `waexport_setup.exe` faylını yükləyin
2. Quraşdırıcını işə salın

## İstifadə

1. WhatsApp-da söhbəti açın → **⋯** → **Söhbəti ixrac et**
2. ZIP faylını yadda saxlayın
3. WA Export-u açın, **FAYL SEÇ** düyməsini basın
4. Lazım gələrsə tərəf adlarını düzəldin
5. **BAXIŞ KEÇİR** ilə önizləməyə baxın
6. **İXRAC ET** ilə qovluğa saxlayın
