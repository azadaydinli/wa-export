WhatsApp söhbətlərini gözəl HTML formatında ixrac edin.

## Xüsusiyyətlər

- WhatsApp ZIP fayllarını oxunaqlı HTML-ə çevirir
- Media fayllarını (şəkil, video, audio, sənəd) saxlayır
- Tarix aralığına görə süzgəc
- Tərəf adlarını və nömrələrini redaktə etmək imkanı
- iOS və Android (Azərbaycan dili) formatlarını dəstəkləyir
- Avtomatik yeniləmə

## Tələblər

- Windows 10 (1903) və ya daha yüksək
- x64 və ARM64 arxitekturası dəstəklənir

## Quraşdırma

1. [Ən son versiyanı yükləyin](https://github.com/azadaydinli/wa-export/releases/latest)
2. `waexport_setup.exe` faylını işə salın və quraşdırın

## İstifadə

1. WhatsApp-da söhbəti açın → **···** → **Export Chat**
2. ZIP faylını yadda saxlayın
3. WA Export-u açın, **FAYL SEÇ** ilə ZIP-i seçin
4. Lazım gələrsə tərəf adlarını düzəldin
5. **BAXIŞ KEÇİR** ilə önizləməyə baxın
6. **İXRAC ET** ilə qovluğa saxlayın
