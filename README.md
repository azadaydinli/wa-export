A Windows desktop app that converts WhatsApp exported ZIP files into a clean, readable HTML format.

## Features

- Converts WhatsApp ZIP exports to HTML
- Preserves media files (photos, videos, audio, documents)
- Filter messages by date range
- Edit display names and phone numbers for both parties
- Supports iOS and Android export formats (including Azerbaijani locale)
- Auto-update on launch

## Requirements

- Windows 10 (1903) or later
- x64 or ARM64

## Installation

1. Download `waexport_setup.exe` from the [latest release](https://github.com/azadaydinli/wa-export/releases/latest)
2. Run the installer

## Usage

1. In WhatsApp, open a chat → **⋯** → **Export Chat**
2. Save the ZIP file
3. Open WA Export and click **SELECT FILE**
4. Adjust display names if needed
5. Click **PREVIEW** to review
6. Click **EXPORT** to save as a folder
