Приложение для Windows, которое конвертирует экспортированные ZIP-архивы WhatsApp в удобный HTML-формат.

## Возможности

- Конвертирует ZIP-экспорт WhatsApp в HTML
- Сохраняет медиафайлы (фото, видео, аудио, документы)
- Фильтрация сообщений по диапазону дат
- Редактирование имён и номеров телефонов участников
- Поддержка форматов iOS и Android (включая азербайджанскую локаль)
- Автоматическое обновление при запуске

## Требования

- Windows 10 (1903) или новее
- x64 или ARM64

## Установка

1. Скачайте `waexport_setup.exe` из [последнего релиза](https://github.com/azadaydinli/wa-export/releases/latest)
2. Запустите установщик

## Использование

1. В WhatsApp откройте чат → **⋯** → **Экспортировать чат**
2. Сохраните ZIP-файл
3. Откройте WA Export и нажмите **ВЫБРАТЬ ФАЙЛ**
4. При необходимости отредактируйте имена участников
5. Нажмите **ПРОСМОТР** для предварительного просмотра
6. Нажмите **ЭКСПОРТ** для сохранения в папку
